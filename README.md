Hola, bienvenido a nuestro Proyecto Interdisciplinario del Primer Cuatrimestre de 2026.     
    
Este proyecto consiste en crear un juego web completo con sistema de registro e inicio de sesión, administración de datos y tabla de puntajes. Para desarrollarlo utilizaremos HTML, CSS, JavaScript, Fetch y una base de datos MySQL.

 # Juego: 
A nuestro grupo (grupo 18) se nos ocurrio hacer un pasapalabra. La idea es que contenga como minimo 10 opciones por letra, ya sea que inicia o contiene.

La meta para este projecto es que aparezca un rosco con las palabras a completar, y que dependiendo si esta bien / mal / todavia no respondida, se le asigne un color diferente (Verde, Rojo, Amarillo). En la pantalla va a aparecer un temporizador, la idea es que la cantidad de tiempo de este este definido por la dificultad elegida por el usuario. Las preguntas tambien serán definidas por la dificultad.

La paleta de colores va a ser comoda para la vista del usuario, dejando a este definirla si quiere colores oscuros o claros.
